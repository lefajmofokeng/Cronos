Position Overview

We are seeking a talented Backend Engineer to join our core engineering team. You will design, develop, and maintain robust server-side applications that power our platform, working with modern technologies to deliver scalable solutions.

Experience 3-7 Years (Senior)
Location SF, CA (Hybrid)
Salary $120k - $180k
Stack Node.js / Python

Key Responsibilities

  • Design RESTful APIs & microservices architecture
  • Optimize database queries for high-volume traffic
  • Write clean, maintainable, self-documenting code
  • Collaborate with Frontend on integration points
  • Implement OWASP security best practices

Requirements

  • BS in Computer Science or related field
  • Proficiency in Node.js, Python, or Java
  • Deep SQL and NoSQL database knowledge
  • AWS/GCP/Azure cloud experience
  • Strong problem-solving methodology
